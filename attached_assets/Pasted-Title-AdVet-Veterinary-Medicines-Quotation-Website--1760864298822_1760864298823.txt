Title: AdVet — Veterinary Medicines Quotation Website

------------------------------------------------------------
🏗️ OVERVIEW
------------------------------------------------------------
Build a responsive bilingual veterinary medicines website called **AdVet** using:
- Plain PHP (no frameworks)
- Vanilla JavaScript (no React, no libraries)
- JSON files for all data (no database)
- Simple Admin Dashboard for content management
- Theme modes: Light / Dark / System auto-detect
- Localization: English (en) + Arabic (ar)
- UI effects and animations inspired by reactbits.dev

Purpose:
Showcase veterinary products and allow users to request quotations (not buy online).  
Admin can manage products, view contact messages, and edit company info/social links.

------------------------------------------------------------
🏠 PUBLIC PAGES
------------------------------------------------------------

1️⃣ HOME PAGE (index.php)
- Hero section with banner, slogan, and “Request Quotation” button.
- Featured Services: loaded from /data/services.json; interactive cards with tilt/hover.
- Featured Products: first few from /data/products.json; cards with hover effects.
- Contact CTA section linking to contact.php.
- Footer with language switcher and theme toggle.

2️⃣ PRODUCTS PAGE (products.php)
- Displays all products (read from products.json)
- Search bar + optional filter by category
- Each product card:
  - Image, Name, Description, Strength, Pack Size
  - “Request Quotation” button → opens quotation form modal
- Quotation form (JS modal):
  - Fields: name, email, phone, quantity, message
  - Submit → POST to /api/quotation_submit.php → save JSON entry
  - Show localized confirmation message.

3️⃣ ABOUT US (about.php)
- Static section: mission, story, values, contact summary.
- Localized (uses strings from /lang).

4️⃣ CONTACT US (contact.php)
- Form: name, email, phone, message
- POST → /api/contact_submit.php
- Save to messages.json
- Optional email notification to admin
- Display localized success message.

🌓 THEME HANDLING
Detect system theme using:
window.matchMedia('(prefers-color-scheme: dark)')

User can toggle theme → saves to localStorage.

CSS variables define theme colors.

Smooth fade transition between modes.

💫 ANIMATIONS (reactbits.dev inspired)
Recreate in vanilla JS:

Fade-in on scroll (IntersectionObserver)

Card hover tilt

Button ripple on click

Lazy image fade-in

Smooth scroll anchors

🔒 SECURITY
Sanitize all inputs using htmlspecialchars().

Validate email and phone formats.

Lock JSON files with flock() when writing.

Use password hashing for admin login.

Optional CAPTCHA on forms.
